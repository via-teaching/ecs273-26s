# Homework 3 — Interactive Stock Dashboard

An interactive data visualization dashboard built with React and D3.js. It displays stock market data for 20 companies through three linked views that all update when a different stock is selected from the dropdown.

## Views

- **Top Left — Line Chart**: Shows Open, High, Low, and Close prices over time for the selected stock. Supports horizontal zoom (scroll wheel) and pan (click and drag).
- **Bottom Left — t-SNE Scatter Plot**: Visualizes all 20 stocks in a 2D space based on price movement similarity. Points are colored by sector. The selected stock is highlighted with a larger dot and label. Supports zoom and pan.
- **Right — News List**: Displays news articles for the selected stock. Click any article to expand and read the full content.

## Setup Instructions

### 1. Navigate to the project folder

```
cd Homework3/xyuzhong/react-js-template
```

### 2. Install dependencies

```
npm install
```

### 3. Copy data files to the public folder (required for the line chart to load CSV data)

```
cp -r data public/
```

### 4. Generate the news JSON file (required for the news panel)

This script reads all the raw `.txt` news files and combines them into a single clean JSON file:

```
node scripts/buildNews.js
```

### 5. Start the development server

```
npm run dev
```

### 6. Open the app in your browser

```
http://localhost:5173
```

## Project Structure

```

├── data/

│   ├── stocknews/        # Raw news .txt files organized by stock ticker
│   ├── tsne.csv          # t-SNE coordinates with ticker and sector labels

│   │   ├── TSNEScatter.jsx   # t-SNE scatter plot with zoom and legend

│   ├── App.jsx               # Main layout and shared state

└── public/               # Static files served by Vite (created in setup step 3)
```

## Notes

- The news content is preprocessed by `buildNews.js` to remove Yahoo Finance boilerplate and sidebar headlines. If news files are updated, re-run `node scripts/buildNews.js` to regenerate `newsdata.json`.
- The t-SNE coordinates in `tsne.csv` were generated using `sklearn.manifold.TSNE` on one year of daily closing price data. t-SNE is non-deterministic, so re-running may produce a different layout.
- All three views are linked through a single `selectedStock` state in `App.jsx`. Changing the dropdown updates all views simultaneously.
