# ECS 273 SQ26 Homework 3 
20 Stocks Info Dashboard (React+TypeScript).

## How to run

**1. Navigate to this folder**

```bash
cd Homework3/cfchou
```

**2. Install Node dependencies**

```bash
npm install
```

**3. Start the dev server**

```bash
npm run dev
```

**4. Open in your browser** 

```bash
usually `http://localhost:5173`
```
